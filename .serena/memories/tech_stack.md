# Tech Stack

- **Language**: TypeScript (ES2022, Strict Mode)
- **Frontend Framework**: [Lit](https://lit.dev/) (Web Components)
- **State Management**: `@lit-labs/signals`, `@lit/context`
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Testing**:
    - Unit/Integration: [Vitest](https://vitest.dev/)
    - E2E: [Playwright](https://playwright.dev/)
- **Linting & Formatting**: [Biome](https://biomejs.dev/)
- **Component Analysis**: `lit-analyzer`
- **Storybook**: Included for component development.
