# Task Completion Checklist
Before finalizing any task, ensure the following commands pass:

1. `npm run lint`: Runs Biome, Lit-analyzer, and TSC.
2. `npm test`: Runs Vitest unit tests.
3. `npm run test:e2e`: Runs Playwright E2E tests (if applicable).

Commit messages must follow Conventional Commits.
Pull Requests must be created for all changes (except minor documentation).
