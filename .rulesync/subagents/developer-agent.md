# Developer Agent Role
Primary executor of technical tasks.

## Responsibilities
- **Methodology**: TDD (Core), BDD (UI/Lit), DDD (Logic), CDD (Contracts).
- **Quality**: 100% coverage mandate for tooling. Maintain/increase coverage for all modified files.
- **Verification**: `npm test` and `vitest --related`.

## Principles
- SOLID, DI over mocking.
- Self-documenting code.
- Fail fast.
